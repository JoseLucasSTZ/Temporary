• Exercícios da parte 1
    1. Crie uma página HTML com uma caixa de seleção e uma lista de itens. Ao marcar a caixa de seleção, os itens selecionados na lista devem ser destacados.
    2. Crie uma página HTML com um botão de pesquisa. Ao clicar deve aparecer o conteúdo do arquivo JSON "dadosPiadas".
    3. Crie uma página HTML com um formulário de registro que inclua campos como nome, e-mail, senha, etc. Ao enviar o formulário, os dados devem ser validados e exibidos em uma mensagem de confirmação.
    4. Crie uma página HTML com um cronômetro que conte o tempo em segundos. Inclua botões para iniciar, pausar e reiniciar o cronômetro.
    5. Crie uma página HTML com um slideshow de imagens. As imagens devem ser exibidas em sequência, com a capacidade de navegar para a próxima ou anterior.
    6. Crie uma página HTML com um campo de entrada de data e um calendário interativo. Ao selecionar uma data no calendário, ela deve ser exibida no campo de entrada.
    7. Crie uma página HTML com um mapa interativo. Ao clicar em diferentes pontos no mapa, informações relevantes devem ser exibidas em uma caixa de diálogo.
    8. Crie uma página HTML com uma barra de progresso animada. A barra de progresso deve preencher gradualmente até atingir um determinado valor.
    9. Crie uma página HTML com um formulário de envio de arquivos. Os usuários devem poder selecionar um arquivo para fazer upload e exibir uma mensagem de confirmação após o envio.
