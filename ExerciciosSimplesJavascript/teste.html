Exercícios

  1. Crie uma página HTML com um botão e um contador. Ao clicar no botão, o contador deve incrementar seu valor em 1.
  2. Crie uma página HTML com uma lista de itens e um botão "Adicionar". Ao clicar no botão, um novo item digitado pelo usuário deve ser adicionado à lista.
  3. Crie uma página HTML com uma imagem e dois botões: "Aumentar" e "Diminuir". Ao clicar nos botões, a imagem deve aumentar ou diminuir de tamanho, respectivamente.
  4. Crie uma página HTML com uma caixa de seleção (checkbox) e um parágrafo. Ao marcar a caixa de seleção, o parágrafo deve mudar sua cor de fundo.
  5. Crie uma página HTML com uma tabela e um botão. Ao clicar no botão, uma nova linha deve ser adicionada à tabela com dados inseridos pelo usuário.
  6. Crie uma página HTML com uma lista de itens e um botão "Remover". Ao clicar no botão, o item selecionado na lista deve ser removido.
  7. Crie uma página HTML com um campo de entrada de texto e um botão. Ao clicar no botão, o texto digitado deve ser exibido em negrito em um parágrafo abaixo.
  8. Crie uma página HTML com um botão e uma div. Ao clicar no botão, a div deve exibir uma animação de transição, como deslizar ou desaparecer gradualmente.
  9. Crie uma página HTML com uma imagem e um botão. Ao clicar no botão, a imagem deve ser substituída por outra imagem.
  10. Crie uma página HTML com um campo de entrada de texto e um botão. Ao clicar no botão, o texto digitado deve ser exibido em um elemento de cabeçalho (h1, h2, h3, etc.) com uma cor aleatória.
